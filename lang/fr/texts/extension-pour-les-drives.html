Description

Drive Off est une extension (open source) pour les navigateurs internet Chrome et Firefox vous permettant de mieux consommer en affichant en temps réel les informations nutritionnelles des produits que vous consultez sur les sites suivants:
  • Auchan
  • Carrefour
  • Biocoop
  • Intermarché
  • Contactez-nous si vous souhaiteriez ajouter la compatibilité avec un Drive ou un site que vous utilisez
Les informations sont affichées sur le bandeau Drive Off, sur les listes de produits (avant de choisir, pour vous permettre de les comparer facilement entre eux), et sur le produit lui même.

Le bandeau

Le bandeau Drive Off affiche les informations suivantes pour un produit donné: Nutri-Score (qualité nutritionnelle d’un produit) Groupe Nova (transformation d’un produit) Eco-Score (impact environnemental d'un produit)

Les données

Drive Off ne pourrait exister sans le projet libre Open Food Facts. Je vous invite à contribuer via les apps Android et iOS et à visiter la version web Open Food Facts

Installer Drive Off

Drive Off est pour désormais disponible sur le Chrome WebStore et le Magasin d'extensions Mozilla pour Firefox. Cliquer ici pour installer l’extension sur Chrome. La version firefox est en cours de développement. Rejoignez le Groupe des testeurs pour partager vos retours.
